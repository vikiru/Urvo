Flip a coin and guess the outcome.

| argument name |                  description                  |
| :-----------: | :-------------------------------------------: |
|   `choice`    | The outcome of the coin flip (heads or tails) |

| Property | Value |
|----------|-------|
| Cooldown | 5 seconds |
| Required User Permissions | None |

## API Reference

No external APIs were used for this command.
