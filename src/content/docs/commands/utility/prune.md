Deletes a specified number of messages in the given channel.

| argument name |           description            |
| :-----------: | :------------------------------: |
|   `amount`    | The number of messages to delete |

| Property | Value |
|----------|-------|
| Cooldown | 10 seconds |
| Required User Permissions | None |

## API Reference

No external APIs were used for this command.
