### ‚ùì Information

Whether you're a fan of anime or manga, Digimon or Disney, or just curious about a term, you'll find everything you need here!

- [Anime](/Urvo/commands/info/anime) - Sends the user information about a queried anime, obtained from MAL.
- [Digimon](/Urvo/commands/info/digimon) - Sends the user information about a queried digimon.
- [Disney](/Urvo/commands/info/disney) - Retrieve information about a specified disney character.
- [Guess User](/Urvo/commands/info/guess-user) - Determine the age, gender, and nationality of a user based on their name.
- [Manga](/Urvo/commands/info/manga) - Sends the user information about a queried manga, obtained from MAL.
- [Manhua](/Urvo/commands/info/manhua) - Sends the user information about a queried manhua, obtained from MAL.
- [Manhwa](/Urvo/commands/info/manhwa) - Sends the user information about a queried manhwa, obtained from MAL.
- [Minecraft](/Urvo/commands/info/minecraft) - Return an image of the minecraft avatar for a specific username/uid.
- [Top Anime](/Urvo/commands/info/top-anime) - Sends the user information about the top 10 anime (all time or airing), obtained from MAL.
- [Top Manga](/Urvo/commands/info/top-manga) - Sends the user information about the top 10 manga (all time or publishing), obtained from MAL.
- [Urban](/Urvo/commands/info/urban) - Provides the definition for a given term from Urban Dictionary.

### üõ°Ô∏è Moderation

Use these commands to manage the server effectively and efficiently!

- [Ban](/Urvo/commands/moderation/ban) - Bans the mentioned user from the server.
- [Kick](/Urvo/commands/moderation/kick) - Kicks the mentioned user from the server.
- [Remove Timeout](/Urvo/commands/moderation/remove-timeout) - Remove the timeout on the mentioned user.
- [Create Role](/Urvo/commands/moderation/role) - Creates a new role based on the inputted name and colour.
- [Timeout](/Urvo/commands/moderation/timeout) - Timeout the mentioned user from the server.
- [Unban](/Urvo/commands/moderation/unban) - Unban the mentioned user from the server.
