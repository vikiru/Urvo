Sends a random xkcd comic in the channel.

| Property | Value |
|----------|-------|
| Cooldown | 10 seconds |
| Required User Permissions | None |

## API Reference

This command was made possible by the following API:

- xkcd Comics ([Website](https://xkcd.com/))
