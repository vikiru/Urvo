## Overview

Sends the user information about a queried digimon.

| argument name |                  description                   |
| :-----------: | :--------------------------------------------: |
|    `query`    | The name of the digimon you wish to search for |

`Cooldown`: 10 seconds

`Required User Permissions`: None
