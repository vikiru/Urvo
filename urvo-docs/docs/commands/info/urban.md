## Overview

Provides the definition for a given term from Urban Dictionary.

| argument name |    description     |
| :-----------: | :----------------: |
|    `term`     | The term to define |

`Cooldown`: 10 seconds

`Required User Permissions`: None

## API Reference

- Urban Dictionary API ([Website](https://www.urbandictionary.com/))
