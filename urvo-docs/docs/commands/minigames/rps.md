## Overview

Play Rock-Paper-Scissors with the bot.

| argument name |                       description                       |
| :-----------: | :-----------------------------------------------------: |
|   `choice`    | The move you would like to play (rock, paper, scissors) |

`Cooldown`: 5 seconds

`Required User Permissions`: None
